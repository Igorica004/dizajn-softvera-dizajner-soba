Domaći zadatak iz Dizajna softvera na Računarskom fakultetu. Domaći rađen sa koleginicom Anastasijom Lazić RN 39/2023. Projekat predstavlja program za dizajniranje enterijera prostorija u kom korisnik sa mišem prevlači elemente, rotira ih, zadaje im dimenzije i vrši ostale transformacije. Fokus projekta je primena dizajn paterna. Program je pisan u Javi i koristi JavaFX za grafički intefejs.
